# Spotify Customization Alternatives to Spicetify

**Recommended:** **Spot** (GTK Spotify client) or **ncspot** (terminal UI)

## Quick Start

### Option 1: Spot (GUI - RECOMMENDED)
```bash
yay -S spot
spot
```
- Native GTK4 Wayland app
- No CEF/GPU issues
- Themes with GTK themes

### Option 2: ncspot (Terminal)
```bash
yay -S ncspot
ncspot
```
- Terminal UI, highly customizable
- Config: `~/.config/ncspot/config.toml`

### Option 3: Spotify Web Player
```bash
firefox https://open.spotify.com
# Install Stylus extension for custom CSS themes
```
