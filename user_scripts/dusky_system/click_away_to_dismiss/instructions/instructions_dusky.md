# 🎯 The "Click-Away to Dismiss" Wayland Protocol (GTK4 Python)

> [!info] **The Goal**
> Wayland's strict security sandboxing prevents apps from seeing mouse clicks outside their own window. This setup uses a tiny, native C-extension to ask the Wayland Compositor (Hyprland) to handle "outside clicks" and dismiss our GTK4 Python popups automatically.

---

## 🧠 1. How It Works (The "Bank Vault" Analogy)

Imagine your Python GTK application is a friendly robot that paints pretty buttons on the screen. However, the Wayland Compositor (Hyprland) is like a highly secure **Bank Vault**.

*   📜 **The XML File (`hyprland-focus-grab-v1.xml`):** This is the **Blueprint** provided by the bank. It outlines the strict rules required to say, *"Alert me if someone touches the floor outside my robot's box."*
*   📠 **`wayland-scanner`:** This is the **Translator**. C-code cannot read XML Blueprints directly. The translator reads the XML and generates two C-readable rulebooks (`.h` and `.c` files) on the fly.
*   🕵️ **The C Script (`dusky.c`):** This is your **Special Agent**. Python sucks at talking to the Bank Vault natively. This Agent reads the translated rulebooks, opens an isolated radio channel to the vault, and listens for the "outside click" alarm.
*   🏭 **`gcc` (The Compiler):** This is the **Bootcamp**. It takes your Special Agent and the translated rulebooks and packs them into a single, deployable gadget: a Shared Object (`.so`) file.
*   📻 **Python `ctypes`:** This is the **Walkie-Talkie**. Python uses this to call the gadget and say, *"I just opened my menu. Watch my back and radio me when you hear the click alarm so I can hide!"*

---

## 📦 2. What Goes Into Your Git Repository?

You **must not** commit the compiled `.so` file. You also do not need to commit the `.h` and `.c` files generated by the scanner. 

You only need to distribute the raw source files so the user's machine can build the gadget using their own system libraries. 

Commit **exactly these two files** to your bare git repo in your target folder (`~/user_scripts/dusky_system/click_away_to_dismiss/`):

1.  `dusky.c`[cite: 3]
2.  `hyprland-focus-grab-v1.xml`[cite: 4]

## 🛠️ 4. The Source Code Reference (`dusky.c`)

If you ever need to recreate the Special Agent, here is the raw C code that goes into `dusky.c`[cite: 3]. It handles creating an isolated communication queue to prevent GTK4 from freezing.

```c
#include <gtk/gtk.h>
#include <gdk/wayland/gdkwayland.h>
#include <wayland-client.h>
#include <pthread.h>
#include <string.h>
#include <stdio.h>
#include "hyprland-focus-grab-v1-client-protocol.h"

static struct hyprland_focus_grab_manager_v1 *grab_manager = NULL;
static struct hyprland_focus_grab_v1 *active_grab = NULL;
static struct wl_event_queue *custom_queue = NULL;
static struct wl_display *global_display = NULL;

typedef void (*ClearedCallback)(void);
static ClearedCallback py_callback = NULL;

// Listen for the grab manager on our isolated queue
static void registry_handler(void *data, struct wl_registry *registry, uint32_t id, const char *interface, uint32_t version) {
    if (strcmp(interface, "hyprland_focus_grab_manager_v1") == 0) {
        grab_manager = wl_registry_bind(registry, id, &hyprland_focus_grab_manager_v1_interface, 1);
    }
}
static void registry_remover(void *data, struct wl_registry *registry, uint32_t id) {}
static const struct wl_registry_listener registry_listener = { &registry_handler, &registry_remover };

// This fires when Hyprland detects an outside click
static void grab_cleared(void *data, struct hyprland_focus_grab_v1 *grab) {
    if (py_callback) {
        py_callback(); // Radio back to Python
    }
}
static const struct hyprland_focus_grab_v1_listener grab_listener = { .cleared = grab_cleared };

// Background thread to dispatch our isolated queue without blocking GTK
static void* dispatch_thread_func(void* arg) {
    while (1) {
        if (wl_display_dispatch_queue(global_display, custom_queue) == -1) {
            break;
        }
    }
    return NULL;
}

void init_wayland_grab(void *gtk_window_ptr, ClearedCallback cb) {
    if (!gtk_window_ptr) return;
    py_callback = cb;

    GtkNative *native = GTK_NATIVE(gtk_window_ptr);
    GdkSurface *gdk_surface = gtk_native_get_surface(native);
    GdkDisplay *gdk_display = gdk_surface_get_display(gdk_surface);

    if (!GDK_IS_WAYLAND_DISPLAY(gdk_display)) {
        fprintf(stderr, "[libwaylandgrab] Error: Not running under Wayland.\n");
        return;
    }

    global_display = gdk_wayland_display_get_wl_display(gdk_display);
    struct wl_surface *wl_surface = gdk_wayland_surface_get_wl_surface(gdk_surface);

    // One-time initialization of the isolated queue
    if (!custom_queue) {
        custom_queue = wl_display_create_queue(global_display);
        struct wl_registry *registry = wl_display_get_registry(global_display);
        
        // Assign the registry proxy to our custom queue to avoid GTK main loop races
        wl_proxy_set_queue((struct wl_proxy *)registry, custom_queue);
        wl_registry_add_listener(registry, &registry_listener, NULL);
        
        // Roundtrip on OUR queue, leaving GTK's events alone
        wl_display_roundtrip_queue(global_display, custom_queue);

        if (!grab_manager) {
            fprintf(stderr, "[libwaylandgrab] Error: hyprland_focus_grab_manager_v1 not supported.\n");
            return;
        }

        // Start background dispatch thread
        pthread_t thread_id;
        pthread_create(&thread_id, NULL, dispatch_thread_func, NULL);
        pthread_detach(thread_id);
    }

    if (grab_manager) {
        if (active_grab) {
            hyprland_focus_grab_v1_destroy(active_grab);
        }
        active_grab = hyprland_focus_grab_manager_v1_create_grab(grab_manager);
        
        // Assign the grab object to our custom queue
        wl_proxy_set_queue((struct wl_proxy *)active_grab, custom_queue);
        
        hyprland_focus_grab_v1_add_listener(active_grab, &grab_listener, NULL);
        hyprland_focus_grab_v1_add_surface(active_grab, wl_surface);
        hyprland_focus_grab_v1_commit(active_grab);
        wl_display_flush(global_display);
    }
}

void destroy_wayland_grab() {
    if (active_grab) {
        hyprland_focus_grab_v1_destroy(active_grab);
        active_grab = NULL;
        wl_display_flush(global_display);
    }
}
```

## 🐍 5. Generic Python GTK4 Integration

Whenever you write a new Python GTK app, you just need two code blocks to plug in your generic tool.

### Block 1: Load the Library (Put near the top of your file)
```python
import ctypes
import os
import logging
from gi.repository import GLib, Gtk

# Point this to our generic compiled tool
_grab_lib_path = os.path.expanduser("~/user_scripts/dusky_system/click_away_to_dismiss/libwaylandgrab.so")
try:
    LIBGRAB = ctypes.CDLL(_grab_lib_path)
    CB_TYPE = ctypes.CFUNCTYPE(None)
except OSError:
    logging.warning(f"Failed to load Wayland Grab Library at {_grab_lib_path}.")
    LIBGRAB = None
```

### Block 2: The Window Boilerplate (Put inside your GTK Window Class)
```python
class MyPopupApp(Gtk.Window): # Or Adw.ApplicationWindow
    def __init__(self, **kwargs):
        super().__init__(**kwargs)
        
        # 1. Protect the C-callback from Python's Garbage Collector
        if LIBGRAB:
            self._grab_cb = CB_TYPE(self._on_grab_cleared)
        else:
            self._grab_cb = None

        # 2. Connect the window signals
        self.connect("notify::visible", self._on_visible_changed)
        self.connect("map", self._on_map)

    # --- NATIVE WAYLAND GRAB LOGIC ---
    def _on_map(self, *args):
        # Attach the grab ONLY after GTK paints the window on the screen
        self._activate_grab()

    def _activate_grab(self):
        if LIBGRAB and self.get_visible() and self._grab_cb:
            # Send the C-memory address of this window to the extension
            window_ptr = ctypes.c_void_p(hash(self))
            LIBGRAB.init_wayland_grab(window_ptr, self._grab_cb)

    def _on_grab_cleared(self):
        # Fired automatically by the C-thread when an outside click happens
        GLib.idle_add(self.set_visible, False)

    def _on_visible_changed(self, *args):
        if self.is_visible():
            pass # The 'map' signal handles the opening grab
        else:
            # Kill the grab when the window hides
            if LIBGRAB:
                LIBGRAB.destroy_wayland_grab()
```
